- Project name
  -     AAI-DOCUPDATE
- Chosen problem
  -     Template Builder. Please review Briefing.xlsx file
- Solution overview
  -     Advisor AI Team uploads a prompt file, this will have details regarding the changes that they want. Along with that, they will upload the client files.
  -     They can upload one or multiple files. Once done, the solution will automatically find all the financial related data that needs to be changed and it will generate a new word file with the timestamp.
  -     Also within the same page, users can see the change history. That means whatever changes the application has carried out before and after delta is visible in the screen.
        1. Open the application url - https://aai-docupdate.lovable.app
        2. In the application page, section Instruction Prompt - requires the prompt based on which you want to generate the file.
        3. You may use the word file - Prompt.docx. Feel free to make any modifications to it.
        4. Click on Generate Updated Document.
        5. Download the Updated Document.
        6. You may also see the modifications by clicking on view changes button.
